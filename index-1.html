Welcome Teachers!

Program Overview

Generation STEM seeks to create an interactive, academic experience for students to learn and explore problem solving skills and techniques utilized in the computer science discipline. Students will learn about teamwork as they work in a group to accomplish a goal. Students will help a Cue robot complete an adventure while learing about computer science and putting STEAM skills into practice.

Lesson Plan

Classes are split into groups of four to five students, with each group using a device equipped with the Cue robot application and a Wonder Workshop Cue Robot. Each robot is assigned a persona, obstacle course, and mission that can be found here. The ultimate goal of the activity is for groups to code their robot’s path through their obstacle course, effectively achieving their narrative’s mission. Implementation

The two engineering design activities were designed to exercise key skills needed to be successful in the field of engineering; such as teamwork, logical problem-solving, and creativity. The activities chosen require various technical and creative skills in the hopes that all students will participate in an activity which intrigues them, plays on their existing skills, and teaches them new ones.

Groups will receive activity instruction cards which detail their robot’s mission. Groups are given a coding instruction card which guides them in writing code to animate their robot. Groups will return to their respective robot and obstacle course to input code on the Cue app. Groups will recieve narractive cards for the beginning and ending of the robots. The materials for each robot is found in the materials page. Encouragement of the engineering design process is told throughout the whole session.
